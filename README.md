## 🎯 Problem Statement
First-time investors are frequently targeted by sophisticated fraud schemes and lack access to transparent, high-performance tools. Existing platforms often have high barriers to entry, opaque operations, and significant execution lag. **ZAP** is built to solve this by providing a **high-speed, transparent, and ultra-secure trading terminal** that leverages **blockchain technology** for absolute synchronization and **AI-powered fraud detection** to protect the assets of the next generation.

---

## 🏗️ Architecture

```
micro-invest/
├── frontend/          # ZAP Terminal (Next.js 15 + Framer Motion + Glassmorphism)
├── backend/           # Node.js + Express + Z-Storage
└── blockchain/        # Hardhat + Solidity Smart Contracts (Sepolia)
```

## 🚀 Elite Tech Stack

### Frontend (Ultra-Premium Terminal)
- **Framework**: Next.js 15 (App Router)
- **Design**: Ultra-Premium Glassmorphism & ZAP Prime Aesthetic
- **Animations**: Framer Motion (Sub-ms perception micro-interactions)
- **Icons**: Lucide Icons (ZAP Vector Set)
- **Theme**: High-Contrast Light/Dark Terminal Modes

### Backend (Execution Node)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Storage**: Z-Storage (Encrypted Local JSON Architecture)
- **Security**: AI Risk Scoring & Real-time Fraud Monitoring

### Blockchain (Settlement Layer)
- **Network**: Ethereum (Sepolia Testnet)
- **Language**: Solidity 0.8.24
- **Framework**: Hardhat
- **Contracts**: Secure Execution Pools & Anti-Fraud Logic

## 🛠️ Installation & Setup

1. **Frontend Terminal**
```bash
cd frontend
npm install
npm run dev
```

2. **Backend Node**
```bash
cd backend
npm install
node server.js
```

3. **Blockchain Audit (Optional)**
```bash
cd blockchain
npx hardhat test
```
